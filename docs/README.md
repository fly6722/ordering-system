# GitHub Pages 前台

這個資料夾是可部署到 GitHub Pages 的顧客點餐前台。

- `index.html`：靜態點餐頁。
- 資料來源：Google Apps Script Web App API。
- 管理後台：仍使用 GAS 的 `?page=admin`。

GitHub Pages 設定建議：

1. 將此專案推到 GitHub。
2. 到 Repository 的 `Settings` -> `Pages`。
3. Source 選 `Deploy from a branch`。
4. Branch 選主要分支，資料夾選 `/docs`。
5. 儲存後使用 GitHub Pages 給你的網址作為顧客點餐網址。
